<br><h3>Execução de testes criados no projeto</h3>

<p><h4>🟢 Projeto training-wheels</h4>

1. Navigate to dir of the project
   cd training-wheels
2. Install the dependencies of the project
   yarn install
3. Run the command
   yarn test

<p><h4>🟡 Projeto samuraibs</h4>

1. Navigate to dir of the project
   cd samuraibs

   yarn install
3. Run the command
   yarn test
